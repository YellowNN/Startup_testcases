Это описание api погодного сервиса.

Задача 2.

Сделать два скрипта(клиент и сервер), а также развернуть локальную базу данных (любую какая нравится)

В базе данных хранятся сущности людей в формате
<Имя><Фамилия><возраст><пол>

Функциональность сервера:
1. Обработать запрос по добавлению сущности в бд
2. Вернуть данные при запросе всех записей.

Функциональность клиента:
1. Пользователь выбирает что сделать - добавить или показать(enter/show)
2. Если show - то клиент выводит в консоль данные полученные с сервера
3. Если enter - ждёт что введёт пользователь и валидирует данные(валидация на твоё усмотрение) после этого клиент отправляет банные на сервер.
